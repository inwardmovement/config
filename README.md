## Keybinds
- `Super+Space` file explorer
- `Super+C` console
- `Super+F` firefox
- `Super+N` notepad
- `Super+B` bloc-note
- `Super+A` à faire
- `Super+S` screenshot
- `Super+D` desktop
- `Super+E` equalizer
- `Super+Tab` menu
- `Ctrl+Alt+C` close
- `Ctrl+Alt+F` full
- `Ctrl+Alt+R` resize
- `Ctrl+Alt+Del` xfce4-session-logout
- `Alt+D` date
